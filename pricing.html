What Maintenance Covers - and Why It Matters

1

API version updates - Meta and TikTok deprecate old API versions. We update your dashboard before it breaks.

2

Token renewal - Instagram API tokens expire every 60 days. We handle renewal so your dashboard never goes dark.

3

Metric accuracy - Platforms redefine how they calculate reach, impressions, and engagement. We keep your numbers trustworthy.

4

New platform features - TikTok Shop, new Reels metrics, Threads - new data becomes available and you'll want it added.

5

Data accuracy audits - Monthly spot-checks that your dashboard matches what you see in-app. Catches drift before it matters.

6

Troubleshooting - If something breaks, you have someone to call - not a new invoice to negotiate.

7

Security - API credentials are sensitive. Maintenance includes credential hygiene, especially if your team changes.

8

Dashboard evolution - Your business grows. New KPIs, new views, new questions. Minor updates are included.
